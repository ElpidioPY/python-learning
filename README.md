## 🎯 Meus Projetos

### Fundamentos & Funções

#### 1. **Adivinhador de Idade** 📅
- **Arquivo:** `01-fundamentos/adivinhador-idade.py`
- **O que faz:** Calcula a idade de uma pessoa baseado em sua data de nascimento
- **Conceitos:** Manipulação de datas, cálculos matemáticos
- **Status:** ✅ Completo

#### 2. **Bom Dia!** 👋
- **Arquivo:** `01-fundamentos/bom-dia.py`
- **O que faz:** Cumprimenta o usuário conforme a hora do dia (bom dia, boa tarde, boa noite)
- **Conceitos:** Condicionais, entrada de dados, lógica
- **Status:** ✅ Completo

### Exercícios com Funções

#### 3. **Exercício Def - Contagem Regressiva** ⏱️
- **Arquivo:** `02-funcoes/contagem-regressiva.py`
- **O que faz:** Realiza uma contagem regressiva usando funções
- **Conceitos:** Definição de funções, loops, parâmetros
- **Status:** ✅ Completo

#### 4. **Maior Número** 📊
- **Arquivo:** `02-funcoes/maior-numero.py`
- **O que faz:** Cria uma lista e encontra o maior número dela
- **Conceitos:** Listas, funções, iteração
- **Status:** ✅ Completo

### Projetos Mais Complexos

#### 5. **Inscrição em Competição** ✋
- **Arquivo:** `04-projetos-integradores/inscricao.py`
- **O que faz:** Verifica se o usuário atende aos requisitos para se inscrever em uma competição
- **Conceitos:** Condicionais, validação de dados, lógica condicional complexa
- **Status:** ✅ Completo

#### 6. **Filho Favorito - Cofre com Senha** 🔐
- **Arquivo:** `04-projetos-integradores/cofre-senha.py`
- **O que faz:** Funciona como um cofre que requer senha correta e ativa uma contagem regressiva ao ser desbloqueado
- **Conceitos:** Validação de entrada, loops, funções, segurança básica
- **Status:** ✅ Completo
